What's included

Eight features, pre-configured

01

CLAUDE.md — Project Memory

Persistent instructions loaded every session. Your stack, conventions, commands, and architecture — Claude remembers it all. The single most important file in any Claude Code project.

02

TASKS.md — Task Tracker

Sprint tracker with session logs. Claude knows what was done, what's in progress, and what's next.

03

Agents — Sub-Agents

Specialized personas with scoped tools. Code reviewer, debugger — consistent behavior. Via .claude/agents/.

04

Rules — Path-Scoped Conventions

Conventions that activate only when relevant. Included rules for database and TypeScript are examples — replace or add your own to match your stack. Only loaded when editing matching files.

05

Skills — Slash Commands

/deploy, /db-migration, /wrap-up — example skills included as starting points. Customize or replace them for your workflow.

06

Hooks — Event-Driven Automation

Auto-format on every edit, block dangerous commands, desktop notifications. Zero-effort guardrails.

07

Settings — Permissions

Explicit allow/deny for tools. rm -rf, sudo, .env writes — blocked by default.

08

MCP Servers — Integrations

Context7, Playwright, Sequential Thinking. External tools pre-wired in .mcp.json.

Project structure

What ships in the template. Customize via Claude — describe your stack and let it adapt everything.

claude-bootstrap-template/
├── CLAUDE.md                          # project memory — loaded every session
├── TASKS.md                           # sprint tracker — read at session start
├── LICENSE
├── .claude/
│   ├── settings.json                  # permissions, hooks, denied patterns
│   ├── agents/

│   │   ├── security-check.sh          # blocks dangerous commands

│   │   ├── database.md                # scoped to migrations + SQL

│       ├── db-migration/SKILL.md      # /db-migration command

├── .mcp.json                          # Context7 + Playwright + Sequential Thinking
